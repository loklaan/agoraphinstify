# Agorophinstify

Web mashup that combines Maps, Crowdsourced Photos and Music to give the illest of chills to a suffering Agorophobe.

## Installation
Install with `npm`:

```shell
$ npm install
```

## Usage
Start a local server using `npm`:

```shell
$ npm start
```
